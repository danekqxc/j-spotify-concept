# 🎵 J-Spotify-Concept

**J-Spotify** — моя первая крупная работа в разработке Android-приложений. Приложение представляет из себя концепт стриминговой платформы, дизайн которой повзаимствован у Spotify и дополнена функциями и механиками других аудиосервисов, а также моим видением архитектуры таких приложений.

## 🌟 Ключевые возможности

### J-Spotify как социальная сеть
Интегрирован полноценный социальный слой, превращая стриминговый сервис в своего рода социальную сеть.
Пользовательские настройки: добавлен личный кабинет пользователя с настройкой визуального отображения главного экрана. В профиле пользователя имеется возможность сменить ник, аватарку; а также добавлена возможность кастомизации главного экрана: добавление своих любимых артистов и своих альбомов.
SoundCloud-style Interaction: реализована система контекстных обсуждений. Каждый трек - это отдельная площадка для комьюнити, где пользователи могут оставлять комментарии и делиться фидбеком в реальном времени через `BottomSheetDialog` интерфейс.


### Динамические видео-холсты
Вместо статичных обложек я реализовал систему иммерсивного погружения:
Visual Engine: Для ключевых композиций внедрена автоматическая подгрузка официальных клипов исполнителей.
Sync Logic: Видеоряд выступает в роли динамического «холста», который бесшовно синхронизируется с локальным аудиопотоком, создавая эффект присутствия без потери качества звука.

### Моя волна
Моя волна — это система персонального вещания, идея которой повзаимствована у Яндекс Музыки. Она формирует бесконечный поток треков, который можно моментально адаптировать под себя с помощью встроенных фильтров по жанрам и настроению.

## 

### Требования
- **Android Studio** (2024.2.1) или новее.
- Установленный **Android SDK 36**.

### Установка
1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/danekqxc/SoundPlayer.git
   ```
2. Откройте проект в Android Studio.
3. Выполните синхронизацию Gradle для загрузки зависимостей (`androidx.media`, `youtube-player`).
4. Выберите конфигурацию запуска `app`.

### Развертывание
Подключите устройство (API 24+) и нажмите **Run**. Для работы видео-холстов требуется активное интернет-соединение.
